# Metik

Dự án tái dựng website [metik.vn](https://metik.vn/) dưới dạng các component React có thể chỉnh sửa bằng Puck Editor.

## Cài đặt

```bash
npm install
```

## Chạy dự án

```bash
npm run dev
```

## Truy cập trình chỉnh sửa (Editor)

Sau khi chạy dự án, mở trình duyệt và truy cập vào địa chỉ:

```
http://localhost:5173/editor
```

Tại đây bạn có thể chỉnh sửa nội dung của 4 trang:

- **Trang chủ** — `/`
- **Giới thiệu** — `/gioi-thieu`
- **Sản phẩm** — `/san-pham`
- **Liên hệ** — `/lien-he`

Sau khi chỉnh sửa xong, nhấn nút **Publish** để lưu thay đổi.
